Stringler, Sayılar, Diziler ve Diziİşlemleri

String

String veri tipi içerisinde metinsel ifadeleri saklar. String veriler değişkenlere girilirken (“) tırnak işareti ya da (‘) kesme işareti kullanılır.

        var ad="emre";
        var soyad='gören';

Length Özelliği – Metnin Uzunluğunu Alma

String değişken ve ifadelerin kaç karakter uzunlukta olduğunu verir.

        var ad="emre";
        var a=ad.length;// a=4Length Özelliği – Metnin Uzunluğunu Alma

indexOf Metodu - Metin İçinde Arama Yapma

Bir string içerisinde başka bir ifadeyi arayarak, o ifade var ise başladığı karakterin numarasını verir. İlk karakterin numarasının 0 olduğu unutulmamalıdır. Eğer aranan ifade bulunamazsa -1 değeri verir.

Aranan ifade tek bir karakter ya da uzun bir ifade olabilir. Aranan ifade birden fazla kez geçiyorsa ilk bulunan eşleşme için sonuç döndürülür.

        var ad="emre";
        var a=a.indexOf("m");//a=1

Slice Metodu – Metnin Parçasını Alma

Başlangıç ve bitiş index numaraları verilerek bir string içerisinden parça almaya yarar.

Aranan ifade tek bir karakter ya da uzun bir ifade olabilir. Aranan ifade birden fazla kez geçiyorsa ilk bulunan eşleşme için sonuç döndürülür.

        var isim = “telefon, tablet, bilgisayar”;
        var a = isim.slice( 9 , 15 );

Aranan ifade tek bir karakter ya da uzun bir ifade olabilir. Aranan ifade birden fazla kez geçiyorsa ilk bulunan eşleşme için sonuç döndürülür.

Sayılar

JavaScript programlama dilinde tek bir sayı türü vardır. Çok büyük veya çok küçük sayılar bilimsel gösterim ile yazılabilir:

        var yıl=1997;
        var pi=3.14; //küsürlü sayı
        var sayi1 = 585e5;    // 58500000
        var sayi2 = 585e-5;   // 0.00585

Diziler

Dizide eleman tanımlamak için kullanılan üç yöntem vardır.

        var sayilar = new Array("sıfır", "bir", "iki", "üç");//Birinci yöntem


        var sayilar = ["sıfır", "bir", "iki", "üç"];//ikinci yöntem


        var sayilar = new Array();
        sayilar[0] = "sıfır";
        sayilar[1] = "bir";         //üçüncü yöntem
        sayilar[2] = "iki";
        sayilar[3] = "üç";

Dizi İşlemleri

push( ) metodu: push dilimizde ittirmek anlamına gelmektedir. JavaScript'te dizinin sonuna eleman eklemek için kullanılır

        var sayilar = ["sıfır", "bir", "iki", "üç"];
        sayilar.push("dört", "beş");

unshift( ) metodu: Bu method JavaScript'te dizinin başına eleman eklemek için kullanılır.

        var sayilar = ["sıfır", "bir", "iki", "üç"];
        sayilar.unshift("eksi üç", "eksi iki", "eksi bir");

sort( ) metodu: Dizi elemanlarını küçükten büyüğe göre sıralamak için kullanılır.

        var sayilar = ["sıfır", "bir", "iki", "üç"];
        document.write(sayilar.sort());

indexof( ) metodu: Diyelim ki bir dizide çok fazla elemanımız var bir elemanı arıyoruz; fakat nerede olduğunu bulamıyoruz. Bunun için indexof( ) fonsiyonunu kullanıyoruz. Bu metod aranan elemanın indeks konumunu verir.

        var sayilar = ["sıfır", "bir", "iki", "üç"];
        document.write(sayilar.indexOf("iki"));
